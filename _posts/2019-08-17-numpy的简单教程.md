### 开始使用numpy      

#### Array（数组篇的学习）：

```python
import numpy as np
# 向量的加法，比如 求0-n平方和立方的和
# 没有numpy的情况下
def sum(n):
  a = range(n)
  b = range(n)
  c = []
  for i in range(n):
    a[i] = a[i] ** 2
    b[i] = b[i] ** 3
    c.append(a[i]+b[i])
  return c

# 如果是用numpy
def sum(n):
    a = numpy.arange(n) ** 2
    b = numpy.arange(n) ** 3
    c = a + b
	return c
```

这样下来，可以看到代码量减少了很多。没有用到for循环等操作，并且运行速度numpy也是更快的。
